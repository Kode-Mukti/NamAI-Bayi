# 06 — User Persona

## Summary

Five primary user personas represent NamAI Bayi's target audience. Each persona reflects real Indonesian parent segments with different needs, technical comfort levels, and decision-making styles.

## Reasoning

A single product cannot serve everyone equally. Personas help prioritize features, tone of voice, AI behavior, and UX decisions. Every design choice should serve at least one persona well.

## Recommendation

### Persona 1: Siti — The Modern Muslim Mom

| Attribute | Detail |
|-----------|--------|
| Age | 27 |
| Location | Jakarta |
| Occupation | Digital Marketer |
| Tech Comfort | High |
| Religion | Islam |
| Culture | Betawi-Javanese mix |
| Needs | Islamic name with modern feel, easy to pronounce internationally |
| Pain Points | Tired of generic Arabic name lists, wants something unique but meaningful |
| Behavior | Active on Instagram and WhatsApp, researches everything online |
| Goal | Find a name that is religious, modern, and Instagram-worthy |

### Persona 2: Made — The Traditionalist Dad

| Attribute | Detail |
|-----------|--------|
| Age | 35 |
| Location | Denpasar, Bali |
| Occupation | Hotel Manager |
| Tech Comfort | Medium |
| Religion | Hindu |
| Culture | Balinese |
| Needs | Traditional Balinese name with proper caste consideration |
| Pain Points | Confused about naming conventions, needs expert guidance |
| Behavior | Asks family first, uses apps as validation |
| Goal | Honor family tradition while giving child a beautiful name |

### Persona 3: Dewi — The Expecting First-Time Mom

| Attribute | Detail |
|-----------|--------|
| Age | 24 |
| Location | Bandung |
| Occupation | Teacher |
| Tech Comfort | Medium |
| Religion | Islam |
| Culture | Sundanese |
| Needs | Sundanese-Javanese blend, strong meaning, nickname potential |
| Pain Points | Overwhelmed by choices, husband disagrees on preferences |
| Behavior | Saves names, discusses with husband, wants partner input |
| Goal | Find a name both parents love with beautiful meaning |

### Persona 4: Alex & Sarah — The International Couple

| Attribute | Detail |
|-----------|--------|
| Age | 31 & 29 |
| Location | Surabaya (Chinese-Indonesian) |
| Occupation | Startup Founder & Designer |
| Tech Comfort | Very High |
| Religion | Christian |
| Culture | Chinese-Indonesian, international |
| Needs | Easy to pronounce globally, works in English and Bahasa |
| Pain Points | Chinese names hard for non-Chinese to pronounce, wants international feel |
| Behavior | Uses AI tools daily, expects high-quality UX |
| Goal | Modern name that works across cultures |

### Persona 5: Ibu Rina — The Grandmother (Influencer)

| Attribute | Detail |
|-----------|--------|
| Age | 55 |
| Location | Yogyakarta |
| Occupation | Retired Teacher |
| Tech Comfort | Low |
| Religion | Islam |
| Culture | Javanese |
| Needs | Traditional Javanese name with deep philosophical meaning |
| Pain Points | Finds apps confusing, prefers human consultation |
| Behavior | Asks family, visits religious leaders, wants printed suggestions |
| Goal | Grandchild has meaningful, traditional Javanese name |

## Implementation

- Design UI for Siti and Dewi (primary personas — highest volume)
- Ensure AI depth for Made and Ibu Rina (cultural expertise)
- Build international pronunciation for Alex & Sarah (differentiator)
- Accessibility features for Ibu Rina (larger text, simpler flow)

## Future Improvement

- Add more personas based on real user data
- Regional-specific onboarding flows
- Age-based UI adaptation
- Partner/couple mode for shared decision-making
