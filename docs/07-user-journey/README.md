# 07 — User Journey

## Summary

The complete user journey from discovery to advocacy. Each stage maps user goals, touchpoints, emotions, and opportunities for delight.

## Reasoning

Understanding the full journey — not just the app experience — ensures we optimize every touchpoint. Parents research baby names for weeks. NamAI Bayi should be present from awareness through decision and sharing.

## Recommendation

### Stage 1: Discovery (Awareness)

| Step | User Action | Touchpoint | Emotion | Opportunity |
|------|------------|------------|---------|-------------|
| 1.1 | Searches "nama bayi" on Google | Google Search | Curious | SEO-optimized content |
| 1.2 | Finds NamAI Bayi article or tool | Blog / Web | Interested | Deep, useful content |
| 1.3 | Sees social media share | Instagram / WA | Intrigued | Viral share preview |
| 1.4 | Reads Play Store description | Play Store | Skeptical | Strong ASO + reviews |

### Stage 2: First Visit (Onboarding)

| Step | User Action | Touchpoint | Emotion | Opportunity |
|------|------------|------------|---------|-------------|
| 2.1 | Opens app/website | Home Screen | Curious | Beautiful, calming design |
| 2.2 | Reads tagline and description | Home Screen | Hopeful | Clear value proposition |
| 2.3 | Taps "Mulai Konsultasi" | Home Screen | Excited | Minimal friction start |
| 2.4 | Enters preferences | Input Form | Engaged | Progress indicator, optional fields |

### Stage 3: AI Consultation (Core Experience)

| Step | User Action | Touchpoint | Emotion | Opportunity |
|------|------------|------------|---------|-------------|
| 3.1 | Submits preferences | Loading Screen | Anticipating | Beautiful loading animation |
| 3.2| AI analyzes and generates | (processing) | Curious | Progressive result reveal |
| 3.3 | Reviews top 5 names | Result Screen | Delighted | Clear, beautiful cards |
| 3.4 | Reads explanation for each | Result Detail | Impressed | Deep, meaningful insights |
| 3.5 | Saves favorite names | Wishlist | Satisfied | One-tap save |

### Stage 4: Evaluation (Decision Making)

| Step | User Action | Touchpoint | Emotion | Opportunity |
|------|------------|------------|---------|-------------|
| 4.1 | Compares saved names | Wishlist | Thoughtful | Side-by-side comparison |
| 4.2 | Shares with spouse | WhatsApp | Collaborative | Beautiful share cards |
| 4.3 | Gets family opinion | Social / WA | Validated | Battle/Vote feature |
| 4.4 | Refines preferences | Re-generate | Determined | Tweak and regenerate |

### Stage 5: Decision (Conversion)

| Step | User Action | Touchpoint | Emotion | Opportunity |
|------|------------|------------|---------|-------------|
| 5.1 | Selects final name | Result Screen | Happy | Celebration animation |
| 5.2 | Saves as chosen name | Wishlist | Decided | Name detail page |
| 5.3 | Shares final decision | Instagram / WA | Proud | Beautiful announcement card |
| 5.4 | Rates the experience | App | Appreciative | Review prompt |

### Stage 6: Advocacy (Retention)

| Step | User Action | Touchpoint | Emotion | Opportunity |
|------|------------|------------|---------|-------------|
| 6.1 | Returns for daily name | Push / Email | Curious | Daily name notification |
| 6.2 | Recommends to friends | Word of mouth | Proud | Referral program |
| 6.3 | Participates in community | Community | Connected | Discussion, voting |
| 6.4 | Explores Premium | Upgrade prompt | Interested | Clear premium value |

## Implementation

- Track journey completion rates with analytics
- Optimize drop-off points (especially Input Form)
- Add micro-interactions at each emotional peak
- A/B test sharing formats for maximum virality

## Future Improvement

- Personalized re-engagement based on journey stage
- Couple Mode journey (two users, one consultation)
- Post-birth journey (name registration, celebration)
- Integration with baby product purchases
