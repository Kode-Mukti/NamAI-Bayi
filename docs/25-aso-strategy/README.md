# 25 — ASO Strategy

## Summary

App Store Optimization strategy for Google Play Store (Android). Focus on Indonesian keywords, cultural relevance, and conversion rate optimization to maximize organic installs.

## Reasoning

Android dominates the Indonesian mobile market with 90%+ market share. Play Store is the primary distribution channel. ASO is the highest-ROI acquisition channel for Android apps — it's free, permanent, and compounds over time.

## Recommendation

### App Title

**Primary:** NamAI Bayi - Konsultan Nama Bayi AI
**Short:** NamAI Bayi

### Subtitle (Play Store 30 chars)

`Cari Nama Bayi dengan AI`

### Keywords (100 chars total)

```
nama bayi, konsultan bayi, nama islami, nama jawa, nama anak, bayi baru lahir, arti nama, rekomendasi nama, nama perempuan, nama laki laki, nama modern
```

### App Description

**Short Description (80 chars):**
Konsultan nama bayi AI untuk orang tua Indonesia. Temukan nama sempurna dengan makna mendalam.

### Rating & Reviews Strategy

| Tactic | Description |
|--------|-------------|
| In-app review prompt | After 3rd consultation (positive UX moment) |
| Review response | Respond to all reviews within 24 hours |
| Review improvement | Address negative reviews in updates |
| Rating prompt | Only show to engaged users (3+ saves) |
| Feedback channel | "Ada saran?" link to internal feedback |

## Implementation

- Prepare all assets before development completion
- Submit to Play Store with full listing 2 weeks before launch
- Run pre-registration campaign (optional)
- Monitor and optimize daily after launch
